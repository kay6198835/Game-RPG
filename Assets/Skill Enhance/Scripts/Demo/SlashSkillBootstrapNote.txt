SlashSkill setup in Unity:

1. Create player GameObject with:
   - PlayerAbilityOwner
   - CharacterStats
   - Health
   - SimpleCharacterMotor
   - AbilitySystem

2. Create condition assets:
   - NotDeadCondition
   - HasEnoughManaCondition

3. Create effect assets:
   - PlayDebugLogEffect
   - LungeForwardEffect
   - DamageInFrontEffect

4. Create AbilityDefinition asset:
   - Id = slash_skill
   - DisplayName = Slash Skill
   - ActivationType = Hold
   - DefaultKey = E
   - Cooldown = 1
   - ManaCost = 10
   - MaxHoldTime = 2

5. Assign conditions and effects to the SlashSkill asset.

6. In AbilitySystem abilityBindings:
   - Slot = Primary
   - Ability = SlashSkill

Behavior:
- Press E: start hold
- Hold up to 2 seconds
- No auto cast at max
- Release E: execute slash
- More hold => more damage + faster/longer lunge
